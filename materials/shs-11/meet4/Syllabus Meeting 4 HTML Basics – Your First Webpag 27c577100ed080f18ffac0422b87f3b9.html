Topic 0: Getting Your Tools Ready

Mission Briefing: Install Your Code Editor

Before we can build a website, we need a special tool called a code editor. Think of it like a super-smart notebook for writing code. We're going to use a very popular and free one called Visual Studio Code (VS Code).

How to Install VS Code

  1. Open your web browser and go to the official VS Code download page: https://code.visualstudio.com/download
  1. Follow the instructions for your computer type.

For Windows 💻:

  • Click the big blue "Windows" button. This will download an installer file (it will end in .exe).
  • Find the downloaded file in your 'Downloads' folder and double-click it.
  • The installation wizard will open. Just click "Next" on all the steps to accept the default settings. It's safe and easy!
  • Once it's finished, VS Code will be installed on your computer.

For Mac 🍎:

  • Click the big blue "Mac" button. This will download a .zip file.
  • Find the downloaded file in your 'Downloads' folder and double-click it to unzip it.
  • You will now see a new application called "Visual Studio Code".
  • Drag this application into your "Applications" folder. That's it!

How to Open and Create Your First HTML File

  1. Open VS Code: Find the Visual Studio Code icon on your computer and open it. You'll see a welcome screen.
  1. Create a New File:
    • Go to the top menu and click File -> New File.
    • An empty, untitled tab will open.
  1. Save the File as HTML: This is a very important step!
    • Go to the top menu and click File -> Save.
    • A save window will pop up.
    • For the File name, type index.html. Using .html at the end is how the computer knows this is a webpage file.
    • Choose a place to save it, like your Desktop or a new folder for your projects.
    • Click "Save".

You're all set! You now have an empty HTML file open in your code editor, ready for you to start building.

Topic 1: What in the World is a Website Made Of?

Mission Briefing: Let's Talk Websites!

Before we start, let's think about the websites we use every day.

  • Opening Discussion: In the chat, share one website you visit a lot. What do you like about it? Is it the text, the images, the links?

Everything you see on those websites—the headings, the paragraphs, the pictures, and the links—is organized by a special language. Today, you're going to learn that language. It's called HTML.

Decoding the Secret Language: What is HTML?

  • HTML stands for HyperText Markup Language.
  • It's the standard language used to create all web pages.
  • Think of it like the skeleton of a webpage. It gives the page its structure.
  • HTML uses "elements" or "tags" to label pieces of content. It tells the browser things like, "this is a heading," "this is a paragraph," or "this is a link."

Let's look at some secret code. What do you think this does?

<h1>Hello World!</h1>
<p>Welcome to my first website.</p>

You probably guessed it! It creates a big heading and a paragraph. You're already starting to understand HTML!

Topic 2: The Skeleton of a Webpage

Every Page Has the Same Bones 🦴

Every single HTML page, no matter how complicated, is built on the same basic skeleton.

A Simple HTML Document Example:

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

Example Explained:

  • <!DOCTYPE html>: This is the very first thing on the page. It tells the web browser, "Hey! This is an HTML5 page."
  • <html>: This is the root or the main container for the entire page. Everything else goes inside it.
  • <head>: This section contains hidden information about the page, like the title. You won't see this content on the main part of the page.
  • <title>: This is the title that shows up in the browser tab at the very top of your screen.
  • <body>: This is the most important part! It's the container for all the visible content—the headings, paragraphs, images, and links that you see on the website.

Topic 3: Understanding HTML Elements & Attributes

What is an HTML Element?

An HTML element is usually made up of a start tag, some content, and an end tag.

<tagname> Content goes here... </tagname>

The element is everything from the start tag to the end tag. Let's break it down:

Start tagElement contentEnd tag
<h1>My First Heading</h1>
<p>My first paragraph.</p>
<br>nonenone
Note: Some HTML elements are "empty" because they don't have any content or an end tag. A common one is <br>, which creates a line break.

Visualizing the Structure

Here’s how all those parts fit together to form a page. The <body> contains everything you see, and the <head> contains the hidden info like the title.

What are HTML Attributes?

Attributes are extra pieces of information you can add to an HTML element. They are like superpowers for your tags!

  • All HTML elements can have attributes.
  • Attributes provide additional information about elements.
  • Attributes are always specified in the start tag.
  • Attributes usually come in name/value pairs like: name="value".

Let's explore some of the most important attributes.

  • The href Attribute for Links The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to.
    <a href="[https://www.w3schools.com](https://www.w3schools.com)">Visit W3Schools</a>
  • The lang Attribute for the Page You should always include the lang attribute inside the <html> tag to declare the language of the Web page. This helps search engines and browsers.
    <!DOCTYPE html>
<html lang="en-US">
<body>
...
</body>
</html>
  • The title Attribute for Tooltips The title attribute defines some extra information about an element. The value of the title attribute will be displayed as a tooltip when you mouse over the element.
    <p title="I'm a tooltip">This is a paragraph.</p>

Important Rules for Attributes

  • Always Use Lowercase Attributes: While TITLE would work, the official recommendation is to always use lowercase, like title.
  • Always Quote Attribute Values: You should always put quotes around the value, like href="some-url.com", not href=some-url.com. This prevents errors, especially if the value has spaces. You can use single ' or double " quotes.

Topic 4: Core Content Tags: Headings & Paragraphs

HTML Headings

HTML headings are titles or subtitles that you want to display on a webpage. They are defined with the <h1> to <h6> tags. <h1> is the most important (and biggest), and <h6> is the least important (and smallest).

  • Headings are Important! Search engines use headings to understand the structure of your page. Use them to organize your content, not just to make text big.
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

HTML Paragraphs

The <p> element defines a paragraph. A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after it.

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

Controlling Space and Lines

  • The Browser Ignores Extra Space: If you put lots of spaces or new lines in your HTML code inside a <p> tag, the browser will ignore them and just show it as a single space.
  • Horizontal Rules (<hr>): The <hr> tag creates a horizontal line, often used to separate content. It's an empty tag.
  • Line Breaks (<br>): The <br> tag creates a line break. Use it if you want a new line without starting a whole new paragraph. It's also an empty tag.
  • The <pre> Element: If you need to display text exactly as it's written in your code, including all spaces and line breaks (like for a poem), use the <pre> tag.
<p>This is<br>a paragraph<br>with line breaks.</p>
<hr>
<pre>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.
</pre>

Topic 5: Formatting & Styling Your Text

You can use special tags to change the appearance and meaning of your text.

Text Formatting Elements

  • <b> - Bold text: <b>This text is bold</b>
  • <strong> - Important text: Typically looks bold, but also tells screen readers the text is important. <strong>This is important!</strong>
  • <i> - Italic text: <i>This text is italic</i>
  • <em> - Emphasized text: Typically looks italic, but tells screen readers to say it with emphasis. <em>This text is emphasized</em>
  • <mark> - Marked text: <mark>Highlights</mark> the text.
  • <del> - Deleted text: Puts a <del>line through</del> the text.
  • <ins> - Inserted text: <ins>Underlines</ins> the text.
  • <sub> - Subscript text: For chemical formulas like H<sub>2</sub>O.
  • <sup> - Superscript text: For footnotes like WWW<sup>[1]</sup>.

The style Attribute

The style attribute lets you add CSS (Cascading Style Sheets) directly to an element. This is a great way to quickly change its appearance.

  • Background Color: style="background-color:powderblue;"
  • Text Color: style="color:red;"
  • Font: style="font-family:courier;"
  • Text Size: style="font-size:160%;"
  • Text Alignment: style="text-align:center;"
  • Borders: style="border:2px solid Tomato;"
<h1 style="color:blue; text-align:center;">This is a blue, centered heading.</h1>
<p style="background-color:Tomato; color:white;">This paragraph has a red background and white text.</p>

Topic 6: Working with Images

The <img> tag is used to embed an image in a web page. It's an empty tag and has two required attributes: src and alt.

Syntax: <img src="url" alt="alternatetext">

Image Attributes Explained

  • The src Attribute (Source): This is the path or URL to the image.
    • Relative URL: An image on your own website (e.g., src="images/my_photo.jpg"). This is the best method.
    • Absolute URL: An image on another website (e.g., src="https://placehold.co/200x200").
  • The alt Attribute (Alternate Text): This text describes the image. It is shown if the image cannot load and is read aloud by screen readers for visually impaired users. It is required!
  • The width and height Attributes: You should always specify the size of your image in pixels. This helps the page load smoothly without flickering.
    <img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

More Image Tricks

  • Image as a Link: To make an image clickable, just wrap the <img> tag inside an <a> tag.
    <a href="page2.html">
  <img src="button.gif" alt="Go to page 2">
</a>
  • Animated Images: HTML supports animated GIFs. Just use the .gif file in your src attribute.

Topic 7: Your First Mission: Build a Personal Profile Page!

Let's Build!

Now it's time to use everything you've learned.

  1. Open VS Code.
  1. Create a new file and save it as index.html.
  1. Type the basic HTML skeleton into your file. Remember to set the language with <html lang="en">.
  1. Inside the <body> tags, create your personal profile. It must include:
    • Your name inside an <h1> tag. Add a style attribute to change its color.
    • A short paragraph about your hobbies using a <p> tag. Use <strong> or <em> on a word you want to emphasize.
    • An image! Use a placeholder from https://placehold.co. Make sure to set the src, alt, width, and height attributes.
    • A link to your favorite website using the <a> tag. Add a title attribute to the link to create a tooltip.
